**PauPins** es una aplicación móvil inspirada en Pinterest, desarrollada como proyecto final para el 8vo semestre. Permite a los usuarios descubrir, guardar y organizar pines(imagenes) mediante la creación de pines y tableros personalizados en tiempo real.


## Tecnologías Empleadas
* **Frontend:** [Flutter](https://flutter.dev/) (Dart) 
* **Backend-as-a-Service:** [Supabase](https://supabase.com/) autenticación y base de datos.
* **Base de Datos:** PostgreSQL.
* **Almacenamiento:** Supabase Storage.
* **Seguridad:** Row Level Security (RLS).


## Instalación y Configuración

### 1. Prerrequisitos
* Tener instalado el SDK de [Flutter](https://docs.flutter.dev/get-started/install).
* Un editor de código.
* Una cuenta en [Supabase](https://supabase.com/).
